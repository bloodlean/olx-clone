
## OLX CLONE
Hello, I'm Amir, I'm a beginner developer and I made an analogue of the OLX website
---

Guide to run

---
1) Install django
```
    pip install django
```
---
2) Clone project from github
```
    git clone https://github.com/bloodlean/olx-clone.git
```
3) Run server
---
# For Windows
```
    winpty python manage.py runserver
```
# For Linux
```
    winpty python manage.py runserver
```
